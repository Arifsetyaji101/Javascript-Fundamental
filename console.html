Console

  • Javascript memiliki fitur logging bernama console
  • Logging adalah mekanisme untuk menampilkan informasi dari aplikasi yang sedang berjalan tanpa harus menganggu alur kerja dan juga interaksi user

Console Method

Console Method Keterangan
console.info(...)
atau console.log(...) 		Memberitahu informasi
console.warn(...) Memberitahu informasi peringatan
console.error(...) Memberitahu informasi peringatan
console.table(...) Memberitahu informasi dalam bentuk table

notes

  • untuk menampilkan console.debug perlu untuk mengubah level pada console untuk bisa menampilkan debug
